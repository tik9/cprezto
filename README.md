
## Own zprezto repo with reference to original zprezto

### Installation:
1. Copy [.zshrc](.zshrc) to your home folder
2. git clone --recursive https://github.com/sorin-ionescu/prezto.git /home/_your-user_/.zprezto

_Optional_ - recommended to see the custom prezto in action:

3. git clone https://github.com/tik9/cprezto.git /home/_your-user_/cprezto
